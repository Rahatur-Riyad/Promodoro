# Pomodoro Timer (Python)

A simple command-line Pomodoro Timer built with Python.  
It helps you manage your study/work sessions using the Pomodoro technique.

## Features
- 25-minute work timer
- 5-minute short break
- 15-minute long break (after 4 cycles)
- Simple terminal-based interface

## How it works
The program runs in cycles:
1. Work session
2. Short break
3. Repeat 4 times
4. Long break

## How to run
```bash
python main.py
